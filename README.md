### Nombre del proyecto: Triple Espresso



Este proyecto es una página web para la cafetería "Triple Espresso". 

La página incluye:

\- Una sección de encabezado con navegación

\- Un formulario de reserva interactivo

\- Un pie de página con enlaces a redes sociales



Tecnologías utilizadas:

\- HTML

\- CSS

\- Flexbox para el diseño

\- Efectos hover y transiciones



Planes de mejora



\- Añadir una sección de menú con los productos de la cafetería

\- Implementar JavaScript para validación del formulario

\- Hacer la página completamente responsiva para dispositivos móviles

\- Agregar animaciones más avanzadas
